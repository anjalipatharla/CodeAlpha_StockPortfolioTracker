# CodeAlpha_StockPortfolioTracker
Stock Portfolio Tracker

Description

Stock Portfolio Tracker is a simple Python application that calculates the total investment value of a stock portfolio based on user-selected stocks and quantities.

Features

- Uses a dictionary to store stock prices
- Accepts stock name and quantity from the user
- Calculates investment value for each stock
- Supports multiple stock entries
- Displays the total portfolio value
- Handles invalid stock names

Technologies Used

- Python
- Dictionary
- While Loop
- Conditional Statements
- User Input/Output

Sample Stocks

Stock| Price
AAPL| 180
TSLA| 250
GOOG| 150

How to Run

1. Open the project folder in VS Code.
2. Open the terminal.
3. Run:

python stock_tracker.py
